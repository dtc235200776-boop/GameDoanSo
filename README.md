# Guess Number Game

## Giới thiệu

* Guess Number Game là trò chơi đoán số được phát triển bằng **C# Console Application**.
* Dự án được thực hiện nhằm luyện tập các kiến thức nền tảng về **C#**, **OOP**, **Clean Code** và **Git/GitHub**.

---

## Mục tiêu

* Rèn luyện tư duy giải quyết bài toán.
* Làm quen với lập trình Console.
* Áp dụng lập trình hướng đối tượng (OOP).
* Biết cách quản lý mã nguồn bằng Git và GitHub.

---

## Tính năng

* Sinh số ngẫu nhiên từ 1 đến 100.
* Người chơi nhập số để dự đoán.
* Gợi ý khi số nhập nhỏ hơn hoặc lớn hơn số bí mật.
* Đếm số lần đoán.
* Kiểm tra dữ liệu nhập bằng `int.TryParse()`.
* Kiểm tra khoảng giá trị hợp lệ.
* Thông báo lỗi khi nhập sai.
* Hỗ trợ chơi lại sau khi kết thúc ván chơi.
* Giao diện Console có màu sắc.
